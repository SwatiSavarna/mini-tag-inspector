Tracking Pixel

01 — How Tracking Pixels Work
1
Marketer embeds invisible image in email
A 1×1 transparent PNG is inserted into HTML content. To the reader, it's completely invisible — zero pixels wide and tall effectively.
<img src="http://localhost:5001/pixel.png?uid=123&campaign=test" width="1" height="1">
2
user opens email / webpage
When the email client or browser renders the HTML, it automatically fetches every image — including the tracking pixel — without any user action.
3
HTTP request reaches the tracker server
The browser sends a GET request. That request involuntarily leaks sensitive headers — no JavaScript needed.
4
Server logs everything & returns pixel
The server records IP address, browser type, OS, language, timestamp, and any custom parameters. It returns the tiny image so the user never suspects anything.
